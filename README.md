## RatEngine - A demo 3D game engine written in Rust and OpenGL

### Reasons
Learn Rust and OpenGL, and have fun.

### Whats implemented:
- [x] OpenGL render
- [x] Camera
- [x] Camera movement
- [x] Model loading
- [x] Textures
- [x] Phong lighting
- [x] Materials
- [x] Commands
- [x] UI
- [ ] Levels
- [ ] Levels editor
- [ ] Skybox
- [ ] Shadows
<br>...
